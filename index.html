GET /s

Fetch football matches, or get specific match by fixtureId

Parameters:
  • date (required): Match date (e.g. 2025-07-22)
  • leagueId (optional): Filter matches by league ID (e.g. 667)
  • fixtureId (optional): Display single match details by fixture ID (e.g. 1371778)
Example:
/s?date=2025-07-22&leagueId=242

GET /f

Get food recipes, or get specific recipe by id

Parameters:
  • id (optional): Get a specific recipe by ID
Example:
/f?id=2

GET /p

Get clothing products, or get a specific product by id

Parameters:
  • id (optional): Get a specific product by ID
Example:
/p?id=5
